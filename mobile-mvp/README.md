# Vibe Idea Mobile MVP

移动端创意生成应用的最小可行产品，专注于通过对话界面激发用户的创意想法。

## 设计理念

遵循"简单实现复杂链路"的原则，通过简洁的对话界面实现复杂的创意激发过程。

### 核心特性

1. **对话式交互** - 移动端聊天界面，用户友好的对话形式
2. **智能感知** - 输入过程中实时感知和反馈
3. **情感连接** - 像素风角色增加互动趣味性
4. **正向引导** - 激励用户不断产生想法
5. **渐进式探索** - 通过对话逐步深入了解用户需求

## 技术架构

### 文件结构
```
mobile-mvp/
├── App.js              # 主应用组件
├── utils/
│   └── ideaEngine.js   # 轻量级想法生成引擎
├── components/         # 可复用UI组件
├── screens/            # 应用屏幕组件
└── assets/             # 静态资源
```

### 核心组件

- **IdeaBuddy** - 像素风角色，提供情感连接
- **MessageBubble** - 消息气泡组件
- **MobileIdeaEngine** - 轻量级想法生成引擎

## 功能说明

### 1. 智能对话
- 用户输入时实时感知关键词
- 根据输入内容调整角色表情和反应
- 生成个性化想法建议

### 2. 像素风角色互动
- 角色表情随对话内容变化
- 思考、兴奋、倾听等不同状态
- 简单动画增强用户体验

### 3. 想法生成
- 基于用户输入分析兴趣领域
- 生成个性化想法建议
- 提供后续探索问题

## 实现细节

### Idea Engine 特性
- 领域分析：自动识别用户兴趣领域
- 模板生成：基于领域生成针对性想法
- 关键词提取：从用户输入中提取关键概念
- 反馈循环：为下一步对话提供建议

### 用户体验
- 简洁的聊天界面
- 即时反馈
- 渐进式信息收集
- 情感化交互
